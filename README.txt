HOW TO OPEN THIS APP?

Step 1: Create your Project Folder (chatapp)
Step 2: Open your terminal and open your existing folder (chatapp)
Step 3: type npm init --yes
Step 3: Install npm i socket.io express
Step 4: Install npm i --save-dev nodemon

Copy paste the ff. files then run it in terminal npm run start and
copy the link http://localhost:3000/ 
After that open it on your browser.
